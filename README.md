<table>
<tr>
<td width="50%" valign="top">

### ‚öôÔ∏è Core

- **Install** paks and uassets into **Legacy** and **IO Store** bundles from folder drops, paks, and archives
- **Install Panel with Per-Mod Settings** ‚Äî custom name, tags, folder destination, compression, Legacy PAK vs IO Store toggle, and Repak toggle per mod
- **Multiple Mod Installations** ‚Äî install several mods at once
- **Organize** pre-packaged mods ‚Äî quickly manage pre-made IO Store bundles
- **Global Clash Detection** ‚Äî resolve mod conflicts with priority selector system
- **Mod Obfuscation/Encryption** ‚Äî protect your mods with AES-256 encryption and asset obfuscation, blocking extraction tools
- **Mod Detection API** ‚Äî auto-detects character, category, mod type, and package type
- **Game Integration** ‚Äî smart crash detection, Steam launch, and mods operations lockdown when game is running
- **Oodle Compression** ‚Äî applied to IO Store bundles to keep your drive happy
- **Mod Sharing** ‚Äî AES-256 encrypted modpack sharing across the globe
- **Parallel Processing Mode** ‚Äî toggle between Normal (50%) and Boost (75%) thread usage for faster mod installations
- **Extract Assets** ‚Äî extract all assets to a chosen folder from a mod bundle
- **Browser Extension Install** ‚Äî receive NexusMods mods from the Repak X browser extension
- **Proper Formatting** of Marvel Rivals proprietary asset formats (Skeletal Mesh, Static Mesh, StringTable and Texture)
- **Utility Tools** ‚Äî Skip Launcher Patch, Character Database, ReCompress, LOD Disabler shortcut
- **Auto-Update Checker** ‚Äî automatically checks and applies new app versions
- **MaterialTag Injection** ‚Äî MaterialTag containers injected through the packing process <sub>(needs the [MaterialTag UE Plugin](https://github.com/XzantGaming/MaterialTagPluginRivals) to create a compatible mod)</sub>
